Qualification File Download Test

This page tests the qualification file download functionality.

Test Cases:
  • Valid Request: Download File (ID: 1)
  • Invalid Request: Test Invalid ID (999)
Expected Behavior:
  • Valid request should download the license file
  • Invalid request should return an error message
  • File should have proper filename and content type
Note: This test assumes you have qualification requests in your database. If no requests exist, the download will fail with "Request not found" error.
